Brief of 5 HTML Tags

1) <body> </body>
It contains all the HTML document contents.
2) <abbr> </abbr>
It helps define abbreviations and acronyms.
3) <ol> </ol>
It defines ordered lists numerically or alphabetically.
4) <ul> </ul>
It defines a list of unordered itenms.
5) <section> </section>
Defines certain sections in a document.
